# Grill-me

Прочитай skill `grilling` и rule `grill-me-gate`. Выполни **строго**.

## Первое сообщение

Только **вопрос 1** в формате grilling. Без плана, PRD, файлов, кода, списка фаз.

## До завершения grill-me запрещено

- Создавать/редактировать любые файлы
- Показывать план фаз или PRD
- Писать «N решений зафиксированы» без ответов пользователя в чате
- Использовать детали из задачи как согласованные без подтверждения

## Завершение

После достаточного числа вопросов (обычно 5–8): «Grill-me завершён. Создать PRD?» — ждать подтверждения.

Затем PRD с **Grill-me log** (каждая строка — реальный ответ из чата).

Run a `/grilling` session now. Question 1 only.
