## Модалки

- «Обратный звонок» (имя + телефон)
- «Оставить заявку» / «Заказать» (имя + телефон + скрытое поле «продукция/тариф»)
- Закрытие по Escape, focus trap, aria-labels

## Анимации (важно)

- Hero: плавное появление заголовка и CTA (CSS или лёгкий JS)
- Карусели: автоплей + стрелки + точки, swipe на touch
- Scroll: лёгкий fade-in секций (prefers-reduced-motion — отключить)
- Hover на кнопках и карточках тарифов
- Без тяжёлых библиотек — предпочтительно CSS + один island при необходимости

## Технические требования

- Astro SSG, `src/pages/index.astro`
- Компоненты в `src/components/`
- Tailwind v4, тёмная спортивная палитра как на референсе
- `<Image>` Astro для фото, WebP где возможно
- Семантика и a11y (rules accessibility)
- Формы: клиентская валидация + mock submit (без бэкенда)
- `pnpm lint && pnpm build` зелёные
- README: как запустить `pnpm dev`
